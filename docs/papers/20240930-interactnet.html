INTERACT-NET: An Interactive Interface for Multimedia Machine Learning

Abstract

INTERACT-NET is a study of interactive human-computer interfaces for use in pipelines of multimedia using machine learning (ML), image processing, and computer graphics.

Author: Alberto Kopiler, Tiago Novello, Guilherme Schardong, Luiz Schirmer, Daniel Perazzo, Luiz Velho
